# Code of Conduct

Be excellent to each other.

## Do

- Treat others with respect
- Give and accept constructive feedback
- Focus on what's best for the project

## Don't

- Harass, insult, or attack others
- Publish private information without consent
- Behave unprofessionally

## Research Ethics

This is dual-use research software. Use responsibly. Follow the license.

## Enforcement

The community self-moderates. Bad actors will be blocked.

---

We're here to build something useful. Keep it professional.
