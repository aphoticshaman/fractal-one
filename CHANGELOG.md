# Changelog

All notable changes to this project will be documented in this file.

## [0.6.0] - 2026-01-14

### Added
- 8-layer AGI architecture (Substrate, Grounding, Proprioception, Containment, Orchestration, Cognition, Alignment, Output)
- Proprioceptive Divergence Testing (PDT) framework
- Axis-P probe system with statistical validation
- Nociception module (pain/damage detection)
- Thermoception module (cognitive heat sensing)
- Vestibular system (balance/stability monitoring)
- Pod methodology (alpha, beta, gamma, delta agents)
- PSAN momentum-gating with golden-ratio scaling
- Drift detection with z-score scoring
- Zero-copy IPC via neuro_link shared memory
- GDELT conflict analysis integration (Shepherd)
- DAG validation CLI

### Changed
- Unified single-binary architecture with feature flags
- Improved clippy compliance
- Sanitized codebase for public release

### Fixed
- SQRT_2 approximation in threshold_detector
- Various clippy warnings (unused imports, clone on Copy, etc.)

## [0.5.0] - 2026-01-07

### Added
- Initial proprioception framework
- Heart timing core
- Cortex monitoring system
- Basic telemetry

This project is now community-maintained. Future changelog entries should be added by contributors.
